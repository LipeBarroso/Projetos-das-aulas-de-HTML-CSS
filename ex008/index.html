Trabalhando com Listas

Listas Ordenadas

Listas ordenadas são aquelas que a ordem faz diferença, essa lista é reprensentada pela tag OL.

E dentro da tag OL temos os itens da lista representado pela tag LI, como por exemplo:

Tarefas do dia a dia:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar HTML e CSS
  5. Fazer almoço
  6. Almoçar
  7. Descansar
  8. Estudar HTML e CSS

Também temos os parametros TYPE por exemplo:

<ol type="1">

Esse parâmetro te dar a opção de sequencia da lista em 1,A,a,I,i.

Temos também a opção de comecar na sua sequencia desejada como por exemplo:

Tarefas do dia a dia:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar HTML e CSS
  5. Fazer almoço
  6. Almoçar
  7. Descansar
  8. Estudar HTML e CSS

Isso acontece quando adicionamos a tag START dentro da tag OL exemplo:<ol start="4">.

Listas Não Ordenadas

Listas não ordenadas são listas onde a sequencia não faz sentido ou seja não precisa enumerar, essa lista é representada pela tag UL.

Coisas que eu gosto de fazer:

  • Jogar bola
  • Jogar Games
  • Tomar uma
  • Ver amigos
  • Cozinhar

Lista de Definiçôes

Lista de definições é como se fosse um dicionario, então adicionamos um termo e a definição dele.

Exemplo:


HTML
Linguagem de marcação de texto para criação de site.
CSS
Linguagm de marcação de design para criação de um site.
JavaScript
Liguagem de programação de interatividade de um site.

No exemplo acima usamos a tag DL e dentro dessa tag temos o parâmetro DT que significa "termo"
e também temos o DD que significa "descrição" como por exemplo:

     
        <dl>
            <dt></dt>
            <dd></dd>   
        </dl>
