Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase temos um termo em destaque usando a tag STRONG, pois ela é um termo semântico.

Italigo ou Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em itálico usando a tag EM, pois ela é um termo semâtico.

Textos Marcados

Podemos também criar um texto marcado usando a tag MARK.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.
Para texto grande usamos a tag BIG (não semântica).
E para texto pequeno usamos a tag SMALL.

Texto Deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado, usando a tag DEL.

Texto Inserido

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag INS, pois ela é um termo semântico.

Existe também o texto sublinhado com a tag U (não semântica).

Texto Sobrescrito

Para inserir coisa do tipo: x20+3 inserimos a tag SUP.

Texto Subscrito

Para inserir coisas do tipo: H2O inserimos a tag SUB.
