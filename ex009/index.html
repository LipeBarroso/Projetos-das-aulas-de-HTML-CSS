Trabalhando com Links

Links Externos

São links para locais externos ou seja que apontam para outros sites (links de saída).

Exemplo:

Você pode acessar os meus repositórios no GitHub.

Você também pode acessar o meu pefil no LinkeDin.

Logo acima usamos a tag A para adicionarmos links externos em algumas palavras da frase.

Usamos também o parâmetro TARGET="_BLANK" REL="EXTERNAL"ele serve para que o usuario abra outra aba no seu navegador e não feixe a do seu site.

Clicando no link do repositorío do GitHub você irá ver que o navegador abrirar outra janela já no link do perfil LinkeDin ele não vai abrir uma janela pois não possue o parâmetro target="blank" dentro da tag A mas abrirar na mesma janela assim fechando automaticamente o site anterior.

Exemplo:

     
    <a href="">repositório no GitHub.</a>

Links Internos

São links para locais internos ou seja que apontam para pagina de seu proprio site ou para outro artigo.

Exemplo:

Clicando aqui você irá para minha pagina 2

E clicando aqui você ira para minha pagina 3
