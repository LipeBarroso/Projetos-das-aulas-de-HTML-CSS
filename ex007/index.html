Segundo Jeff Noble, no seu livro HTML para leigos:

As diferenças entre elementos inline e um bloco de texto é importante.Os elementos de HTML neste capítolo descrevem os blocos de texto.

Nessa frase acima usamos a tag BLOCKQUOTE, ela e uma tag que desloca uma parte de um texto horizontalmente sendo uma citação de uma pessoa ou autor.
E também usamos um parâmetro dentro da tag BLOCKQUOTE que foi a CITE como por exemplo:

        
            <blockquotecite="(link do livro)">

Esse parâmetro tem como objetivo citar o livro, e os mecanismos de busca vai indentificar que essa frase está referente a este livro.
